Обзор

Этот проект использует модели глубокого обучения для обработки аудио и изображений. Он включает в себя несколько API для предсказания классов аудиосигналов и обнаружения объектов на изображениях.

API Эндпоинты

/api/dense

Метод: POST
Описание: Принимает аудиофайлы и возвращает предсказания классов.

Пример запроса

POST /api/dense
Content-Type: multipart/form-data

Files: audio1.wav, audio2.wav

Пример ответа

{
    "predictions": [["Guitar_Sound"], ["Drum_Sound"]],
    "uploaded_files": ["audio1.wav", "audio2.wav"]
}

/api/cnn

Метод: POST
Описание: Принимает аудиофайлы и использует CNN для классификации.

Пример запроса

POST /api/cnn
Content-Type: multipart/form-data

Files: audio1.wav

Пример ответа

{
    "predictions": [["Piano_Sound"]],
    "uploaded_files": ["audio1.wav"]
}

/api/yolov5

Метод: POST
Описание: Принимает изображения и возвращает данные об обнаруженных объектах.

Пример запроса

POST /api/yolov5
Content-Type: multipart/form-data

Files: image1.jpg

Пример ответа

{
    "predictions": [
        {
            "file": "image1.jpg",
            "results": [{"name": "Piano", "confidence": 0.85}]
        }
    ],
    "uploaded_files": ["image1.jpg"]
}

Установка

Чтобы установить и запустить проект, выполните следующие шаги:

  1. Клонируйте репозиторий: git clone <url>
  2. Установите зависимости: pip install -r requirements.txt
  3. Запустите приложение: flask run
